# Obsidian 팀 운영 볼트

## 목적
노션 비용 없이 파일 기반으로 팀 일정/업체/장비/이슈/Todo를 관리합니다.

## 폴더 구조
- `vendors/` 업체 문서
- `equipments/` 장비 문서
- `issues/` 이슈 문서
- `todos/이름/` 개인 할일 문서
- `reports/` 데일리/위클리 브리핑
- `templates/` 템플릿

## 권장 플러그인
- **Tasks** (할일 쿼리)
- **Dataview** (YAML 기반 집계)
- **Templater** (빠른 문서 생성)

## 빠른 사용법
1. 템플릿으로 업체/장비/이슈/할일 문서 생성
2. YAML 필드를 채움 (`vendor`, `equipment`, `assignee`, `status`, `due` 등)
3. `dashboard.md`에서 전체 현황 확인

## 팀원
- 오세영
- 임혜란
- 고봉석
- 김민준
