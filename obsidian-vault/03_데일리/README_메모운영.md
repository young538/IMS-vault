# 메모 운영 (임시 전면 개편)

## 원칙
- 일정/스케줄 관리 대신 **메모 중심**으로 운영
- 옵시디언 포맷(폴더 체계) 유지
- 메모는 가능하면 **옵시디언 + 노션 동시 기록**

## 빠른 기록 명령
```bash
cd /home/young/.openclaw/workspace
scripts/memo_dual_write.sh "제목" "메모 본문" "업무"
```

- 분류 기본값: `기록`
- 분류 예시: `업무`, `아이디어`, `기록`, `기타`

## 기록 위치
- 옵시디언: `03_데일리/YYYY-MM-DD.md`의 `## 메모` 섹션
- 노션 DB: `메모 - 오세영`
